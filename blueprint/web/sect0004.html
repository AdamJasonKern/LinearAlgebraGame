3 VectorSpaces

3.1 ZeroScalarMultiplication

Definition 8 Vector Space
✓
#

Lean declarations

  • Game.VectorSpace

A vector space is a space over a field K with an abelian group V. It has four main properties:

  • smul_add: ∀ (a : K) (x y : V), a • (x + y) = a • x + a • y

  • add_smul: ∀ (a b : K) (x : V), (a + b) • x = a • x + b • x

  • mul_smul: ∀ (a b : K) (x : V), (a * b) • x = a • (b • x)

  • one_smul: ∀ (x : V), (1 : K) • x = x

Definition 12 Subspace

  • Game.Subspace

A subspace is a subset of a vector space that acts similarly to a vector space itself. It has three main properties:

  • Nonempty: W.Nonempty

  • Closure_Under_Addition: ∀ (x y : V), x ∈ W → y ∈ W → x + y ∈ W

  • Closure_Under_Scalar_Multiplication: ∀ (a : K) (x : V), x ∈ W → a • x ∈ W
